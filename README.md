# WEB CLOSET

A modern fashion marketplace aggregator built with React and TypeScript.

## Features

- Clean and modern UI design
- Responsive layout for all devices
- Search functionality across multiple marketplaces
- Popular items showcase
- Partner logos section
- Fully typed with TypeScript

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
  components/
    Header.tsx          # Header with logo and navigation
    Hero.tsx            # Hero section with search
    PopularItems.tsx    # Popular items carousel
    PartnerLogos.tsx    # Partner logos section
    Footer.tsx          # Footer with contact info
  App.tsx               # Main app component
  main.tsx              # Entry point
  index.css             # Global styles
  App.css               # App-level styles
```
